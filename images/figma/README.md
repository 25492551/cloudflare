## 사용 가이드

### 웹사이트에 적용하기

1. **로고 사용**
   - `HUMMAN RESEARCH.svg` - 메인 로고 (SVG 권장, 확장 가능)

2. **섹션 태그**
   - `SCIENCE.svg`, `PLATFORM.svg`, `R&D.svg`, `EXPERIENCE.svg` - 카드 섹션 태그

3. **타이틀 이미지**
   - Hero 섹션: `RECODING YOUR VITALITY.svg`
   - 섹션 제목: `THE FUTURE OF ENDOGENOUS LIPID SIGNALING.svg`

4. **그래픽 요소**
   - 배경/장식 요소로 사용: Vector, Rectangle, Polygon, Ellipse 등

### 최적화 권장사항

1. **SVG 우선 사용**
   - 가능한 경우 SVG 형식 사용 (확장 가능, 파일 크기 작음)
   - PNG는 투명 배경이 필요한 경우에만 사용

2. **이미지 최적화**
   - 웹 사용 전 이미지 압축 권장
   - WebP 형식으로 변환 고려 (파일 크기 30-50% 감소)

3. **파일명 정리**
   - 필요시 더 명확한 파일명으로 변경
   - 카테고리별 폴더 구조로 재구성 가능
